# 📄 AI Resume Screening System

## 🚀 Overview

An AI-powered Resume Screening System that automatically analyzes resumes and ranks candidates based on job requirements using Natural Language Processing (NLP).

---

## 🎯 Features

* 📂 Upload resumes (PDF/Text)
* 🧠 NLP-based keyword extraction
* 🎯 Job-role matching system
* 📊 Resume ranking & scoring
* ⚡ Fast automated filtering

---

## 🛠️ Tech Stack

* Python
* NLP (TF-IDF / Scikit-learn)
* Pandas, NumPy
* Streamlit (UI)
* Matplotlib (visualization)

---

## 📂 Project Structure

```
resume_screening_system/
│
├── app.py                # Streamlit UI
├── model/                # Saved NLP model
├── data/                 # Resume dataset
├── notebooks/            # Experimentation
├── requirements.txt

```

---

## ▶️ How to Run

### 1. Install dependencies

```
pip install -r requirements.txt
```

### 2. Run application

```
streamlit run app.py
```
